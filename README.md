# Hotelinventoryapp

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 16.2.1.

## Running this project

1. Run the api 
    1. Use the terminal to go to ../hotelapi
    2. `npm run start`
2. Start angular on this project
    1. `ng serve -o` -o will open a browser
